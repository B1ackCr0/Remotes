# Netflix (Web)
Netflix remote control.

## Features
*  Launch Netflix site
*  Lower volume
*  Mute volume
*  Raise volume
*  Pause playback
*  Toggle playback state
*  Navigate left
*  Select current item
*  Navigate right
*  Seek forward
*  Seek backward
*  Fullscreen view
*  Windowed view
