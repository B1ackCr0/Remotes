# J River
J River media center remote.

## Features
*  Send raw command to J River
*  Start playback
*  Pause playback
*  Toggle playback state
*  Previous track
*  Next track
*  Seek backward
*  Seek forward
*  Navigate home
*  Toggle fullscreen
*  Stop playback
*  Raise volume
*  Lower volume

## Platforms
* Windows
