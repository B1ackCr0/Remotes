# Hulu (Desktop)
Hulu desktop player remote.

## Features
*  Focus Hulu application
*  Launch Hulu application
*  Navigate up
*  Navigate left
*  Select current
*  Navigate right
*  Navigate down
*  Navigate back
*  Raise volume
*  Lower volume

## Platforms
* Windows
