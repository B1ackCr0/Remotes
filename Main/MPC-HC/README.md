# MPC-HC
Media Player Classic Home Cinema Remote.

## Features
*  Launch MPCHC application
*  Run command
*  Fullscreen
*  Move up
*  Move down
*  Move left
*  Move right
*  Move back
*  Select
*  Play Pause track
*  Pause track
*  Play track
*  Stop track
*  Forward track
*  Rewind track
*  Next track
*  Prevous track
*  Go to Home
*  Open title
*  Open chapters
*  Volume up
*  Volume down
*  Mute Volume
*  Jump Back 5s
*  Jump Forward 5s

## Platforms
* Windows
