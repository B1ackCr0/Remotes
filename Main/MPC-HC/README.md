# MPC-HC
Media Player Classic Home Cinema Remote.

## Features
*  Forward/Rewind track
*  Fullscreen
*  Go to Home
*  Jump Back 5s
*  Jump Forward 5s
*  Move up/down/left/right/back
*  Next/Prevous track
*  Play/Pause/Stop track
*  Volume up/down/mute
