# Media Portal
Media Portal remote

## Features
*  Launch MediaPortal application
*  Stop playback
*  Navigate up
*  Toggle playback state
*  Navigate left
*  Select current
*  Navigate right
*  Previous track
*  Navigate down
*  Next track
*  Seek backward
*  Seek forward
*  Navigate back
*  Raise volume
*  Lower volume
*  Mute volume
*  Navigate home
*  Toggle fullscreen
*  Show information

## Platforms
* Windows
