# Media Monkey
Media Monkey Remote.

## Features
*  Launch iTunes application
*  Play Pause track
*  Play track
*  Pause track
*  Stop track
*  Next track
*  Previous Track
*  Mute Volume
*  Set Volume
*  Volume up
*  Volume down
*  Set Position
*  Rate Up
*  Rate Down

* Windows
