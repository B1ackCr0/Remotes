# USB-UIRT
Helper remote for USB-UIRT.

## Features
*  Learn code
*  Transmit code
*  Learn code (Pronto format)
*  Learn code (UUIRT format)
*  Transmit saved code (Pronto format)
*  Transmit saved code (UUIRT format)
*  Transmit code to computer clipboard
*  Transmit UUIRT code
*  Transmit Pronto code

## Platforms
* Windows
