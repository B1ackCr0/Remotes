# Spotify
Spotify basic media remote control.

## Features
*  Update status information
*  Start playback
*  Pause playback
*  Toggle playback state
*  Lower volume
*  Raise volume
*  Mute volume
*  Next track
*  Previous track
*  Stop playback
*  Launch Spotify application
*  Send raw command to Spotify
