# Plex (Keyboard)
Control Plex Home Theater

## Features
*  Focus Plex application
*  Launch Plex application
*  Navigate left
*  Navigate down
*  Navigate right
*  Navigate up
*  Select
*  Navigate back
*  Play pause
*  Seek forward
*  Seek rewind
*  Navigate home
*  Stop playback
*  Show menu
*  Next item
*  Previous item
*  Play current item
*  Show OSD
*  Show info
