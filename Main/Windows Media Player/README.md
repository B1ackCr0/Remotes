# Windows Media Player
Windows Media Player remote control.

## Features
*  Launch Windows Media Player
*  Send command to program
*  Toggle fullscreen
*  Mute volume
*  Lower volume
*  Raise volume
*  Previous track
*  Next track
*  Stop playback
*  Toggle play/pause

## Platforms
* Windows
