# BSPlayer
BSPlayer remote control.

## Features
*  Launch BSPlayer application
*  Send raw command to BSPlayer
*  Zoom in
*  Zoom out
*  Normal zoom
*  Toggle playback state
*  Toggle fullscreen
*  Raise volume
*  Lower volume
*  Navigate left
*  Navigate right
*  Navigate up
*  Navigate down
*  Previous track
*  Next track
*  Stop playback
*  Fast rewind
*  Rewind
*  Forward
*  Fast forward
*  Show DVD subtitles
*  Show DVD title
*  Show DVD languages
*  Previous DVD chapter
*  Show DVD chapters
*  Next DVD chapter

## Platforms
* Windows
