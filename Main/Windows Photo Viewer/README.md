# Windows Photo Viewer
Windows Photo Viewer remote.

## Features
*  Start slide show
*  Rotate photo counter-clockwise
*  Rotate photo clockwise
*  Zoom out
*  Zoom in
*  End slide show
*  Previous photo
*  Next photo

## Platforms
* Windows
