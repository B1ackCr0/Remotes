# Windows Media Center
Windows Media Center remote control.

## Features
*  Launch WMC application
*  Lower volume
*  Mute volume
*  Raise volume
*  Pause playback
*  Navigate up
*  Start playback
*  Navigate left
*  Select current item
*  Navigate right
*  Stop playback
*  Navigate down
*  Navigate back
*  Previous item
*  Go home
*  Record
*  Next item
*  Previous channel
*  Next channel

## Platforms
* Windows
