# TellStick
TellStick remote

## Features
*  Toggle deivce by ID
*  Toggle device by name
*  Toggle all devices
*  Turn on all devices
*  Turn off all devices
*  Turn on by ID
*  Turn on by name
*  Turn off device by ID
*  Turn off device by name
*  Dim device by ID
*  Dim device by name
*  Dim all devices
*  List all devices (id, name, model)
*  Find device ID by name

## Platforms
* Windows
