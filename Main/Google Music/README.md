# Google Music
Google Music remote.

## Features
*  Close
*  Decrease rating
*  Increase rating
*  Instant
*  Lower volume
*  Navigate down
*  Navigate home
*  Navigate up
*  Next track
*  Previous track
*  Raise volume
*  Search
*  Seek backward
*  Seek forward
*  Select current item
*  Toggle playback state
*  Toggle repeat
*  Toggle shuffle
