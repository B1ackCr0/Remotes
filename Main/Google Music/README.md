# Google Music
Google Music remote.

## Features
*  Change track (next, previous)
*  Navigate (up, down, left, right)
*  Open Search
*  Rating (decrease, increase)
*  Seek (forward, backward)
*  Select current item
*  Toggle (play, repeat, shuffle)
*  Volume (raise, lower)
