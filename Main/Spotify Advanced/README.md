# Spotify Advanced
Spotify remote with playlists.

## Features
*  Change Position
*  Change Volume
*  Next/Previous track
*  Shuffle/Repeting
*  Play/Pause track
*  Show playlists (currently unstable)
*  Search artist/album/track (currently unstable)

## Platforms
* Windows
* Mac OSX
