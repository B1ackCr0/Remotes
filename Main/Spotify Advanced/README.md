# Spotify Advanced
Spotify remote with playlists.

## Features
*  Launch Spotify application
*  Change position
*  Change volume
*  Next track
*  Previous track
*  Set repeting
*  Set not repeting
*  Play track
*  Pause track
*  Play Pause track
*  Set Suffle
*  Set not Shuffle
*  Send raw command to Spotify
*  Start playback
*  Pause playback
*  Toggle playback state
*  Toggle Shuffle 
*  Toggle Repeat 
*  Change Volume
*  Change Position

## Platforms
* Windows
* Mac OSX
