# MusicBee
MusicBee media player remote.

## Features
*  Focus Chrome application
*  Launch Chrome application
*  Toggle playback state
*  Stop playback
*  Next track
*  Previous track
*  Lower volume
*  Raise volume
*  Toggle mute volume

## Platforms
* Windows
