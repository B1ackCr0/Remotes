# YouTube (Web)
YouTube remote control.

## Features
*  Launch YouTube
*  Lower volume
*  Raise volume
*  Rewind
*  Fast forward
*  Play previous item
*  Play next item
*  Enter fullscreen
*  Exit fullscreen
*  Toggle play/pause
