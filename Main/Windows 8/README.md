# Windows 8
Windows 8 navigation.

## Features
*  Navigate right/left/up/down
*  Select current item
*  Show start screen
*  Show apps
*  Show charms
*  Snap app to left/right
*  Open run
*  Open explorer
*  Navigate back
*  Zoom in/out
