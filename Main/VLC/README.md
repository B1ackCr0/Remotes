# VLC
Basic VLC remote.

## Features
*  Toggle play/pause
*  Raise/Lower/Mute volume
*  Seek forward/backward
*  Toggle fullscreen
*  Next/Previous playlist item
*  Stop playback
*  Jump back 10 seconds
*  Jump forward 10 seconds
*  Step back 5 seconds
*  Step forward 5 seconds

## Platforms
* Mac OSX
* Windows
