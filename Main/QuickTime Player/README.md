# QuickTime Player
Remote control for QuickTime player.

## Features
*  Launch QuickTime application
*  Toggle play/pause
*  Start/resume playback
*  Pause playback
*  Stop playback
*  Seek backwards
*  Seek forwards
*  Raise volume
*  Lower volume
*  Toggle mute volume
*  Seek to position

## Platforms
* Mac OSX
