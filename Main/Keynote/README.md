# Keynote
Remote control for the Keynote app.

## Features
*  Start slide show
*  Show slide switcher
*  Cancel slide switcher
*  Accept slide switcher
*  Slide switcher forward
*  Slide switcher backward
*  End slide show
*  Previous slide
*  Next slide

## Platforms
* Mac OSX
