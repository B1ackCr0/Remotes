# foobar2000
foobar2000 media remote

## Features
*  Update state
*  Focus foobar2000 application
*  Launch program
*  Lower volume
*  Mute volume
*  Raise volume
*  Previous track
*  Next track
*  Stop playback
*  Toggle playback state
*  Send raw command

## Platforms
* Windows
