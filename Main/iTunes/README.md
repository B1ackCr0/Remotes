# iTunes
The iTunes Remote provides advanced control of iTunes on Windows and Mac. It includes standard playback control and playlist access.

## Features
* Current playing information
* Volume slider
* Position slider
* Toggle playback state
* Next/previous track
* Volume control
* Toggle fullscreen
* Shuffle and repeat
* Playlist picker
