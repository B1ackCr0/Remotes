# SoundCloud
Remote for SoundCloud.com

## Features
*  Launch SoundCloud
*  Lower system volume
*  Mute system volume
*  Raise system volume
*  Previous track
*  Next track
*  Toggle play & pause
*  Repeat playing song
*  Seek Forward
*  Seek Backward
*  Like playing song
