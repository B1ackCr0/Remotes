# Kodi Advanced
Kodi web interface remote.

## Features
*  Toggle play/pause/stop
*  Play nextprevious item
*  Rewind/Fast forward
*  Raise/Lower/Mute volume
*  Navigate up/left/right/down
*  Toggle Context menu/OSD/Fullscreen
