# Beamer
Beamer remote control.

## Features
*  Update status information
*  Launch Beamer application
*  Previous movie
*  Next movie
*  Stop playback
*  Toggle play/pause state
*  Set position
*  Open file manager

## Platforms
* Mac OSX
