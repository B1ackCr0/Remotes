# Beamer
Remote control for the Beamer Mac app when streaming video from your Mac to Apple TV.  
http://beamer-app.com/

## Features
*  Next movie
*  Previous movie
*  Stop playback
*  Toggle play/pause state
*  Set position

## Platforms
* Mac OSX
