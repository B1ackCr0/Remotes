# Daum PotPlayer
Daum PotPlayer remote control.

## Features
*  Jump back 30 seconds
*  Jump back 5 seconds
*  Jump forward 30 seconds
*  Jump forward 5 seconds
*  Lower volume
*  Mute volume
*  Next track
*  Pause or unpause playback
*  Previous track
*  Raise volume
*  Start playback
*  Stop playback
*  Toggle fullscreen
*  Toggle play/pause state
*  Toggle playlist
