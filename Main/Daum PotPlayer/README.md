# Daum PotPlayer
Daum PotPlayer remote control.

## Features
*  Jump back 30 seconds
*  Jump back 5 seconds
*  Jump forward 30 seconds
*  Jump forward 5 seconds
*  Raise/Lower/Mute volume
*  Next/Previous track
*  Pause or unpause playback
*  Start/Stop playback
*  Toggle fullscreen
*  Toggle playlist
