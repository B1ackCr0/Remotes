# Boxee (Keyboard)
Boxee remote using key shortcuts.

## Features
*  Fast forward
*  Lower volume
*  Mute volume
*  Navigate back
*  Navigate down
*  Navigate home
*  Navigate left
*  Navigate right
*  Navigate up
*  Raise volume
*  Rewind
*  Stop playback
*  Toggle fullscreen
*  Toggle playback state

## Platforms
* Windows
