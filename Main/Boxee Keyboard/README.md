# Boxee (Keyboard)
Boxee remote using key shortcuts.

## Features
*  Focus Boxee application
*  Launch Boxee application
*  Lower volume
*  Mute volume
*  Raise volume
*  Rewind
*  Fast forward
*  Stop playback
*  Toggle playback state
*  Navigate up
*  Navigate down
*  Navigate left
*  Navigate back
*  Navigate right
*  Toggle fullscreen
*  Navigate home
*  Select current item

## Platforms
* Windows
