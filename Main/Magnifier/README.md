# Magnifier
Control the Windows magnifier utility.

## Features
*  Zoom in
*  Zoom out
*  Pan Up
*  Pan left
*  Pan right
*  Pan down
*  Windows key
*  Exit
*  Invert
*  Full screen mode
*  Lense mode
*  Docked mode

## Platforms
* Windows
