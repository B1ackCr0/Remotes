# XBMC (Keyboard)
XBMC remote using key shortcuts.

## Features
*  Toggle play/pause/stop
*  Raise/lower/mute volume
*  Navigate up/left/right/down
*  Select current item
*  Play next/previous item
*  Fast forward/rewind
