# VLC Advanced
VLC web interface remote.

## Features
*  Change position
*  Seek backwards
*  Seek forwards
*  Launch VLC application
*  Toggle play/pause
*  Start playback
*  Resume playback
*  Pause playback
*  Stop playback
*  Play next item
*  Play previous item
*  Toggle shuffle
*  Toggle loop
*  Toggle repeat
*  Raise volume
*  Lower volume
*  Mute volume
*  Change volume
*  Toggle fullscreen
