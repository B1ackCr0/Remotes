# Boxee Advanced
Boxee Web interface remote.

## Features
*  Toggle playback state
*  Start playback
*  Pause playback
*  Stop playback
*  Fast forward
*  Rewind
*  Navigate up
*  Navigate down
*  Navigate left
*  Navigate right
*  Select current item
*  Navigate back
*  Leave DVD
*  Set volume
*  Raise volume
*  Lower volume
*  Set position
*  Mute volume
*  Toggle fullscreen
*  Previous media
*  Next media
